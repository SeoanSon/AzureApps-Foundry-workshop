# 1. 사전 준비

> MFA 등록을 위해 모바일 앱스토어 (혹은 플레이스토어)에서 Microsoft Authenticator 설치가 필요합니다.
> 

![image.png](./images/image.png)

1. 사전에 전달 받은 계정 정보를 확인합니다.
2. 브라우저에서 [Azure Portal](https://portal.azure.com/)에 접속합니다. (InPrivate window를 사용하는 것이 좋습니다.)
3. 전달 받은 계정 정보를 사용하여 로그인합니다.
    - ID : <alias>@MngEnvMCAP639727.onmicrosoft.com
4. 다음과 같은 MFA 등록 화면이 나오면 `Next` 버튼을 클릭합니다.
    
    ![image.png](./images/image%201.png)
    
5. 모바일 Authenticator 앱에서 브라우저에 표시되는 QR 코드를 찍어 MFA 등록을 진행합니다.
    
    ![image.png](./images/image%202.png)
    
6. MFA가 정상적으로 등록되면 임시 비밀번호를 업데이트합니다.
    
    ![image.png](./images/image%203.png)
    
7. 정상적으로 로그인되면, Azure Portal 을 확인할 수 있습니다.
8. 상단 `톱니 바퀴 모양 포털 설정` 버튼을 클릭합니다.
    
    ![image.png](./images/image%204.png)
    
9. 왼쪽 `Language + region` 메뉴를 클릭하여 Language를 `한국어`로 설정 후, Apply 버튼을 클릭합니다.
    
    ![image.png](./images/image%205.png)
